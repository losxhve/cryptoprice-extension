# CryptoPrice - Chrome 浏览器插件

实时查看 BTC/ETH/SOL 等15个主流币种价格的 Chrome 扩展。

## 功能
- 一键查看15个主流币种实时价格
- 24小时涨跌幅
- 点击价格一键复制
- 搜索过滤
- 自动刷新（每分钟）
- 链接到 CryptoAlerts 告警系统

## 上架 Chrome 商店
1. 打开 https://chrome.google.com/webstore/devconsole
2. 注册开发者（$5一次性费用）
3. 上传 `crypto-tracker-extension/` 目录
4. 定价：免费+捐赠 或 $1.99

## 本地安装（开发模式）
1. 打开 Chrome → chrome://extensions
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `crypto-tracker-extension/` 目录
